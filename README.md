# Tank-game-Python-project
Tank Game is a simple Python game built using Pygame. Players control a tank, move around, and shoot at enemy tanks. The game tracks your score and provides a straightforward, fun arcade experience.

Note: All images and sounds used in this project are for fun and personal use only. This project is for learning purposes and does not intend to infringe on any copyrights.

# Features
1. Move your tank around freely.
2. Shoot at enemy tanks.
3. Track your score.
4. Simple, easy-to-play arcade mechanics.
5. Optional: You can view mechanics of the game via the settings icon.

# Requirements
- Python 3.x
- Pygame library
- Install Pygame using pip: pip install pygame

# How to Run
1. Download .py file.
2. Extract the zip files. Make sure you have the Tank-Game.py, Sounds Folder, and Images Folder.
3. Run the File(make sure you installed the requirements).
